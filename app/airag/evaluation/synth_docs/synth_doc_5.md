# Closely Related Concepts

## 1. BATNA versus reservation point

A BATNA is the action taken if no deal is reached, whereas a reservation point is the worst deal still acceptable.

## 2. target versus reservation point

A target is the desired realistic result, whereas a reservation point is the acceptance boundary.

## 3. position versus interest

A position states what a party demands, whereas an interest explains the need motivating that demand.

## 4. mediation versus arbitration

A mediator facilitates voluntary agreement, whereas an arbitrator issues a decision under the agreed process.

## 5. distributive versus integrative

Distributive bargaining allocates a fixed pool, whereas integrative bargaining searches for trades that can expand joint value.

## 6. lead time versus cycle time

Lead time begins at the request, whereas cycle time begins only when active work starts.

## 7. risk versus issue

A risk is an uncertain future event, whereas an issue is a problem that has already occurred.

## 8. accountable versus responsible

The responsible person performs the work, whereas the accountable person owns the final outcome and approval.

## 9. milestone versus task

A milestone has zero duration and marks an event, whereas a task consumes time and produces work.

## 10. verification versus validation

Verification checks whether a product was built to specification, whereas validation checks whether it meets user needs.
